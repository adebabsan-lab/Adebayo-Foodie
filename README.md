# Adebayo's Nigerian Foodie

A luxury Nigerian food blog built with Flask.

## Features

* Home Page
* Recipe Showcase
* Food Gallery
* Nigerian Food Stories
* Contact Page
* Responsive Design
* Render Deployment Ready

## Installation

Clone repository:

git clone https://github.com/yourusername/adebayo-nigerian-food-blog.git


cd adebayo-nigerian-food-blog

Install dependencies:

pip install -r requirements.txt

Run application:

python app.py

browser:

http://127.0.0.1:5000

## Deployment

Deployed directly on Render using the included render.yaml file.
